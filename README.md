# MegaBlog

A full-stack blog application where users can sign up, log in, create, edit, and view blog posts with a rich text editor.

## Tech Stack

- **Frontend**: React, React Router
- **State Management**: Redux Toolkit
- **Styling**: Tailwind CSS
- **Backend**: Appwrite (for auth, database, and storage)
- **Editor**: TinyMCE
- **Build Tool**: Vite
